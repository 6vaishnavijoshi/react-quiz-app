# React Quiz App

A simple Quiz Application built using React.js.

## Features

* Display multiple-choice questions
* Answer selection
* Score calculation
* Correct answer checking
* Quiz completion screen
* Restart Quiz functionality
* Responsive and clean UI

## Technologies Used

* React.js
* JavaScript (ES6)
* CSS3
* Vite

## Project Structure

```text
src/
├── App.jsx
├── App.css
├── main.jsx
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/6vaishnavijoshi/react-quiz-app.git
```

2. Navigate to the project folder

```bash
cd react-quiz-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
